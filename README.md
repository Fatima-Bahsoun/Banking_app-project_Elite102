# Banking_app-project_Elite102
Simple banking app project using tkinter, mysgl, and python.
The ration of the app is 16:9.

Feutures:
    Create account and set intial balance;
    deposit money into account;
    withdraw money from account;
    Check balance.
Setup:
    The following must be downloaded:
        Python3 
        MySQL 
        MySQL python connector
    In MySQL:
        create the testdb database
        fill in the password and check the info in main.py to match
    Run the code from the banking_app.py file using the line command: python3 banking_app.py
